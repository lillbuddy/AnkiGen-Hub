2. 重新命名檔案與選取 (0 張)

「檔案名稱」預設取自原始檔名，可以直接改成這張圖片的正式名稱（例如：封面.jpg）。就算你沒有加副檔名，系統也會自動幫你補上原始的圖片格式（.jpg / .png...），確保匯入 Anki 後圖片一定讀得到。瀏覽器沒辦法直接改掉你電腦裡的原始檔案，所以改好之後，請按右上角「下載重新命名的圖片」，會下載內容相同、檔名已經改好的圖片副本，之後把這些下載下來的檔案複製進 Anki 媒體庫即可。取消勾選「納入匯出」可以先略過還沒確認的圖片。

下一步的匯出模式（選擇題 CSV / Image Occlusion CSV）都會直接使用這裡改好的檔案名稱來參照圖片，所以請先把名字確認好，再繼續下一步。

Anki 萬用選擇題模板程式碼

只要在 Anki 中新增一個自訂卡片類型，將以下程式碼貼入對應位置，就能呈現完全一樣的效果！（跟「選擇題卡片生成器」用的是同一個筆記類型，如果你已經在那邊建立過，這裡不用重複建立。）

在 Anki 新增卡片類型 (Note Type) 時，請建立且精確命名以下 10 個欄位：

Question 存放題目內容（含圖片標籤）
OptionA 選項 A 內容
OptionB 選項 B 內容
OptionC 選項 C 內容
OptionD 選項 D 內容
OptionE 選項 E 內容 (選填，無則留空)
OptionF 選項 F 內容 (選填，無則留空)
Answer 答案英文字母，如 A 或 A, C
IsMultiple 若是多選題，隨便填入任何字(如y)，單選則留空
Explanation 卡片解析內容 (選填)
正面模板 (Front Template)
背面模板 (Back Template)
共享 CSS 樣式 (Styling CSS)

匯入設定提醒

Image Occlusion 模式用的是 Anki 內建的 Image Occlusion 筆記類型，不需要自己建立模板。匯入 CSV 時，請在 Anki 的匯入畫面把「筆記類型」選成 Image Occlusion，並確認欄位對應順序如下：

1. Occlusion 已帶入「尚未框選：檔名」佔位文字（不能留空，否則 Anki 會把整列當空白筆記跳過不匯入）；匯入後請在 Anki 打開筆記，用遮罩編輯器畫出實際遮蓋範圍
2. Image 已自動帶入 <img src="檔名">
3. Header 已自動帶入檔名（去除副檔名）
4. Back Extra 已自動帶入你填的備註
5. Comments 留空
