# AnkiGen Hub - 醫學選擇題卡片生成器 & 模板庫

一個網頁工具，能將 Markdown 格式的考卷透過 AI 快速解析，自動轉換為具備美觀互動效果的 Anki 選擇題卡片，並提供萬用卡片模板。

## 🚀 如何啟動
1. 下載或定位到此專案資料夾。
2. 直接雙擊 `index.html` 檔案，即可在你的瀏覽器中開啟使用。
3. 不需要安裝任何軟體、不需要架設伺服器；使用 AI 智慧解析功能時，需要自備 Gemini API Key。

---

## 📝 Markdown 考卷格式規範
為了能讓智慧解析器順利分析，請確保你的考卷文字符合以下基本格式（網頁內建「載入範例」按鈕，可一鍵查看）：

```markdown
1. 關於二尖瓣狹窄的敘述，下列何者錯誤？
A. 最常見的原因是風濕熱
B. 心尖處可聽到舒張期心雜音
C. 常合併心房顫動
D. 第一心音會變弱
答案：D
解析：二尖瓣狹窄時，第一心音通常會變強（Loud S1）...
```

* **題號**：每題開頭必須是數字加點、頓號或右括弧（例如 `1.` 或 `2、`）。
* **選項**：必須是字母加上點、頓號或右括弧（例如 `A.` 或 `B)`），每行一個選項。最多支援到 F（6 個選項），最少要有 A 和 B。選項如果少於 5 個（例如只有 A-D），系統匯出到 Anki 後會自動適應並隱藏多餘按鈕。
* **答案**：請以 `答案：` 或 `Ans:` 開頭。如果是多選題，答案直接連寫或用逗號隔開即可（如 `答案：A, B, C` 或 `答案：AC`），系統會自動識別為多選題。
* **解析**：以 `解析：` 或 `Explanation:` 開頭，可選填。

## ⚙️ Anki 軟體設定指南（只需設定一次）

要讓卡片呈現網頁模擬器中精美的互動與高亮效果，請按照以下步驟在電腦版 Anki 設定你的卡片種類：

### 步驟 1. 新增卡片類型 (Note Type)
1. 開啟 Anki，點選上方工具列的 **管理 -> 工具 -> 卡片類型管理**。
2. 點選 **新增** -> 選擇 **加入:基本型** -> 將其命名為 `Medical_MultipleChoice`（或任何你喜歡的名字）。

### 步驟 2. 設定卡片欄位 (Fields)
1. 選取你剛新增的 `Medical_MultipleChoice`，點選右側的 **欄位** 按鈕。
2. 修改並新增欄位，請**務必完全符合且精確命名**以下 10 個欄位（順序無所謂，但名字要一致）：
   * `Question`
   * `OptionA`
   * `OptionB`
   * `OptionC`
   * `OptionD`
   * `OptionE`
   * `OptionF`
   * `Answer`
   * `IsMultiple`
   * `Explanation`

### 步驟 3. 貼入模板程式碼 (Templates)
1. 在卡片類型管理視窗中，點選右側的 **卡片** 按鈕（進入模板編輯畫面）。
2. 在網頁工具右下角的「Anki 模板程式碼」區域，複製對應內容並貼上：
   * 網頁中的 **正面 HTML** 貼到 Anki 的 **正面模板 (Front Template)** 欄位。
   * 網頁中的 **背面 HTML** 貼到 Anki 的 **背面模板 (Back Template)** 欄位。
   * 網頁中的 **共享 CSS** 貼到 Anki 的 **樣式表 (Styling CSS)** 欄位。
3. 點選 **儲存**。

## 📥 如何將題目匯入 Anki

1. 在網頁左側貼上你的 Markdown 考卷，點選 **智慧解析考卷**。
2. 在下方的預覽表格中確認或修改題目、選項、正確答案與解析。
3. 點選 **匯出 Anki 匯入檔 (CSV)**，這將下載一個 `.csv` 檔案（實際上是用 Tab 分隔的文字檔，最不容易出錯）。
4. 打開 Anki，點選下方的 **匯入檔案**，選擇剛才下載的 `.csv` 檔案。
5. **關鍵匯入設定（非常重要）**：
   * **卡片類型**：選取你剛建立的 `Medical_MultipleChoice`。
   * **牌組**：選擇你想放入的牌組。
   * **欄位分隔字元**：請務必下拉選擇 **「逗號 (Comma)」**。
   * **欄位配對**：Anki 會自動將 CSV 的 Question 配對到 Question 欄位、OptionA 配對到 OptionA 欄位...依此類推。
   * **允許在欄位中使用 HTML**：**請務必勾選此項**！因為本工具會自動將換行轉換為 `<br>` 標籤，勾選後才能在 Anki 中正常換行。
6. 點選 **匯入**。大功告成！你現在擁有超精美、具備正面點選互動、背面自動綠色高亮答案的醫學選擇題卡片了！
